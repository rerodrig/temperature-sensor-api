# Temperature Sensor API
## Prerequisites
```
Java 1.8 (jdk)
Docker
Docker Compose
```
### Build project
```sh
/temperature-sensor-api$ ./gradlew clean build
```

### Run locally
```sh
/temperature-sensor-api/local$ ./start.sh
/temperature-sensor-api$ ./gradlew bootRun
```
 ### Swagger Docs
 http://localhost:8080/swagger-ui.html
